# TD-CamTrack
Touchdesigner osc-arkit-to-virtual-camera tool

Meant to be used for virtual production environments, interactive experiences, VFX & mixed reality.

To use this, install any ARKit-based tool that can track the iPhone’s spatial data and send it out via OSC. Connect the TD server to the same network as the phone, enter the network parameters, enjoy.

Built & Tested using iPhone 12 Pro & ZIG SIM PRO.
